### 构建

由于自动化部署的项目，没有部署到理想的路，因此在这里执行远程命令，将项目移动正确的位置

![](https://cdn.jsdelivr.net/gh/TheFoxFairy/notebook-picgo@master/img/20200922225958.png)

* SSH SITE:在前面已经配置好了，选中即可
* Command:这里是远程在服务器命令行中所执行的命令

```
利用cp命令进行覆盖和rm -rf 删除
```
